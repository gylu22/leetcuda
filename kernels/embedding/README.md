## embedding 函数
torch.nn.functions.embedding 函数
```python 
from torch.nn.functional import embedding
"""
args:
  input (Tensor) – Long tensor containing the indices to extract
  weight (Tensor) – Embedding matrix with shape (num_embeddings, embedding_dim)
...
"""
```
* weight 参数是一个二维矩阵，行数是字典大小，列数是每个词向量的维度。[num_embeddings, embedding_dim]
* input 输入是一个矩阵，行数是batch_size，列数是每个batch的词向量个数。[batch_size, seq_index] 
* <font color=red>本示例中input 维度是一维 </font>

`weight size = [10,5] input size = [2,3]，output：[2,3,5]`

## 说明
#define LDST128BITS(value) (reinterpret_cast<float4 *>(&(value))[0])

这个宏的作用是将一个浮点数的地址重新解释为float4类型的指针，然后取该指针指向的第一个float4元素。float4是CUDA内置的向量类型，包含4个单精度浮点数。因此，这个宏实际上是将一个浮点变量（作为起始地址）转换为一个float4类型的向量，从而可以一次性处理4个浮点数。
```yaml
weight数组（源）：
+---+---+---+---+---+---+---+---+
| f0| f1| f2| f3| f4| f5| f6| f7| ...   (每个f是一个float)
+---+---+---+---+---+---+---+---+
      ↑
      offset + 4*tx 位置开始
      视为一个float4结构：{f0, f1, f2, f3}

output数组（目标）：
+---+---+---+---+---+---+---+---+
| f0| f1| f2| f3| f4| f5| f6| f7| ...
+---+---+---+---+---+---+---+---+
      ↑
      bx*emb_size + 4*tx 位置开始
      写入一个float4：{f0, f1, f2, f3}
```
